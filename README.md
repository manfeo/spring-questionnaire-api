# Spring Questionnaire API

**Демо-проект** - упрощённый бэкенд-сервис для управления анкетами пользователей с поддержкой полнотекстового поиска и загрузки файлов.
Представлен для демонстрации навыков работы со стеком Spring Boot, Hibernate Search.

## Технологии и Стек

*   **Язык:** Java 17
*   **Фреймворк:** Spring Boot 3.3.3 (Web, Data JPA, Validation)
*   **База данных:** H2 Database (in-memory / file mode)
*   **Поиск:** Hibernate Search 7 + Apache Lucene 9
    *   Полнотекстовый поиск (Full-text search)
    *   Русская морфология (Russian Light Stemming)
    *   Автодополнение (N-grams)
*   **Документация API:** Springdoc OpenAPI 3 (Swagger UI)
*   **Маппинг:** MapStruct
*   **Тестирование:** JUnit 5, Mockito, Spring Boot Test
*   **Утилиты:** Lombok, Maven Wrapper

## Ключевые возможности

1. **CRUD операций с анкетами:** Создание, чтение, обновление и удаление профилей.
2. **Производительный поиск:** Реализован через **Apache Lucene** для высокой скорости работы (вместо медленного `LIKE` в SQL).
    *   Индексирование происходит асинхронно на старте (`MassIndexer`).
    *   Автоматическое обновление индекса при изменении данных.
3. **Загрузка файлов:** Поддержка `multipart/form-data` для прикрепления изображений.
4. **Валидация:** Строгая проверка входных данных через Bean Validation API.
5. **REST API:** Полное соответствие стандартам REST, корректные HTTP-статусы, обработка ошибок.

## Как запустить

### 1. Требования
*   Java JDK 17 или выше
*   Maven (или использовать встроенный `mvnw`)

### 2. Запуск
Откройте терминал в корне проекта и выполните:
mvn spring-boot:run

Документация доступна по адресу:
http://localhost:8080/swagger-ui.html
